변수명 만들기

변수는 밑줄 문자(_) 혹은 영문자로 시작해야 한다.

  1. 수치형 자료: 정수형(int), 실수형(float), 복소수형(complex)

      • int(일반적인 정수형): 숫자 그대로 입력

WHILE 문

x = 0
while x < 10:
    x = x+1
    if x < 3:

<관계 연산자>

시퀀스 자료에 대한 이해

시퀀스 자료는 어떠한 객체가 순서를 갖고 나열이 되는 것을 말한다. 문자열, 리스트, 튜플이 그러한 시퀀스 자료형이다.

특성 설명
1. 인덱싱 인덱스를 통해 해당 값에 접근 할 수 있다.
인덱스는 0부터 시작
2. 슬라이싱 특정 구간의 값을 취할 수 있다.
구간은 인덱스 범위를 정의
3. 연결 + 연산자를 이용하여 두 시퀀스 자료를 연결하여
새로운 시퀀스 자료를 형성한다.
4. 반복 * 연산자를 이용하여 시퀀스 자료를 여러번 반복하여
새로운 시퀀스 자료를 형성한다.
5. 멤버체크 ‘in’ 키워드를 이용하여
특정 값이 시퀀스에 속해 있는지 확인
6. 크기정보 len()을 활용하여 시퀀스 자료의 크기를 알 수 있다.
시퀀스 자료의 크기는 문자열은 문자의 개수

# 키워드 인자가 불명확한 경우가 있을 경우 **를 사용하면 함수 내부에서 '사전'으로 처리하게 된다.

파이썬 모듈과 패키지 이해

모듈은 흔히 말하는 라이브러리의 개념과 일치한다.
패키지 모듈은 ‘계측적인 디렉토리’ 형태로 구성한 파이썬 패키지이다. 디렉토리를 파이썬 패키지로 인식을 하려면 계측적으로 이루어져 있는 각 디렉토리마다 __init__.py라는 이름의 파이썬 파일이 있어야 한다. 해당 py의 내용은 간단한 내용만 있어도 된다.

다시 말하여, 모듈이 계층적으로 있을 경우 이들을 구분하기 위한 요소로서 __init__.py가 있고 이들이 일종의 구분을 하는 역할을 한다.

오류가 발생할 가능성이 큰 부분에 대하여 try~except 문을 실시하여, 오류가 발생했을 경우 단순히 stop 하는 것이 아니라 except 문으로 넘어가서 해당 작업을 수행하게 된다.

    print('안녕')

except:

    print('예외가 발생하지 않았다')

else 구문까지 추가하는 경우 try 문이 모두 TRUE일 경우 else 구문을 반환한다.

try ~ except ~ finally 구문

try:
    print('안녕')
    print(var)
except:
    print('예외가 발생')
finally:  
    print('무조건 수행')

try~except의 구문과 관계 없이 무조건 finally 이하 구문을 수행을 하게 된다.

try:

except Exception as e:
    print('e')

에러가 발생할 경우 해당 error에 대한 msg를 e로 넘길 수 있는 구문이다.

time 모듈의 localtime()은 대한민국의 현재시각을 time.struct_time 형식의 데이터로 리턴을 한다. 이는 설정된 지역에 따라 해당 지역의 시간을 반환한다. time.struct_time의 형식은 9개의 속성을 가지고 있으며, 각 속성에 대한 값들은 속성 이름이나 인덱스로 접근이 가능하다.

인덱스 속성이름 설명
0 tm_year 현재년도(yyyy)
1 tm_mon 현재 월 (1 ~ 12)
2 tm_mday 현재 날짜(1 ~ 31)
3 tm_hour 현재 시간 (0 ~ 23)
4 tm_min 현재 분 (0 ~ 59)
5 tm_sec 현재 초 (0 ~ 61; 61은 윤초일 때 사용)
6 tm_wday 현재 요일 (0 ~ 6; 0은 월요일)
7 tm_yday 1월 1일부터 현재까지의 날짜수 (1 ~ 365)

따라서 t=localtime()으로 지정을 한다면 t[0] 혹은 t.tm_year 을 통하여 현재년도를 구할 수 있다.

%S 초를 나타냄
%x 현재 날짜를 월/일/년으로 나타냄 (06/28/2018)
%X 현재 시간을 시:분:초로 나타냄 (15:53:24)

디렉토리를 하나의 ZIP파일로 압축하기

from zipfile import *
import os

def compressALL(zipname, folder):
    with ZipFile(zipname,'w') as ziph:
        for dirname, subdirs, files in os.walk(folder):
            for file in files:
                ziph.write(os.path.join(dirname, file))

folder = 'tmp'
zipname = folder + '.zip'
compressALL(zipname, folder)

os.walk의 경우는 주어진 디렉토리를 모두 탐색하는데 사용된다. 즉, 여러 하위 구조를 갖고 있는 디렉토리에 대해 walk(폴더명)을 입력하면 해당 폴더명 아래에 있는 디렉토리와 파일들을 탐색하고 결과를 리스트로 저장을 하게 된다.

탐색된 서브 디렉토리가 있을 경우, 해당 디렉토리와 파일들을 동일하게 탐색하고 결과를 저장한다. walk()의 결과는 다음과 같이 반환한다.
[('탐색 디렉토리 이름', ['찾은 서브 디렉토리 이름'...],['찾은 파일 이름...])...]
